# NETFLIX-CONTENT-STRATEGY-ANALYSIS---POWER-BI
This dashboard analyzes content from Netflix (2008–2021) to understand trends in content mix, genres, ratings, release patterns, and geographic distribution. The insights help identify strategic opportunities to improve content investment decisions, strengthen audience engagement, and support subscriber retention.

**Project Overview**
This project analyzes the Netflix Titles dataset (2000–2021) to identify key content trends and develop data-driven strategic recommendations that could help improve:
1) Subscriber growth
2) Content engagement
3) Revenue performance

**Tools Used:-**
Power BI, Power Query (Data Cleaning & Transformation), DAX (Data Analysis Expressions), Data Visualization & Business Intelligence

**Data Preparation:-**
Data cleaning and transformation were performed using Power Query, including:

1) Handling missing values
2) Standardizing country and genre fields
3) Splitting multi-value column (genres)
4) Creating structured tables for analysis
5) A data model was built in Power BI using:
    - Fact table: Netflix Titles dataset
    - Dimension tables: Shows and Genre bridge table
6) Manage relationship for data visualization.
