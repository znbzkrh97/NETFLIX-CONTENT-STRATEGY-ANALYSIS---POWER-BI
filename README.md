# NETFLIX-CONTENT-STRATEGY-ANALYSIS---POWER-BI

**Project Overview**
This project analyzes the Netflix Titles dataset (2000–2021) to identify key content trends and develop data-driven strategic recommendations that could help improve:
1) Subscriber growth
2) Content engagement
3) Revenue performance

**Tools Used:-**
Power BI, Power Query (Data Cleaning & Transformation), DAX (Data Analysis Expressions), Data Visualization & Business Intelligence

**Data Preparation:-**
Data cleaning and transformation were performed using Power Query, including:

1) Handling missing values
2) Standardizing country and genre fields
3) Splitting multi-value column (genres)
4) Creating structured tables for analysis
5) A data model was built in Power BI using:
    - Fact table: Netflix Titles dataset
    - Dimension tables: Shows and Genre bridge table
6) Manage relationship for data visualization. 

**Dashboard Structure**
This repository includes a 5-page Power BI dashboard that presents the analysis, insights, and strategic implications derived from the Netflix content dataset.

Page 1 – Title
This page introduces the objective of the dashboard, providing context for the analysis. It also includes the author information and reference sources used in the project.

Page 2 – Overview
This page presents a high-level overview of the dataset, including key trends, visual summaries, and distribution patterns. The goal is to help users quickly understand the overall composition of the Netflix content library, including content types, release trends, and general patterns in the data.

Page 3 – Detailed Analysis
This section provides a deeper analytical view of the data using interactive filters and slicers. It examines:
- Content rating distributions
- Top genres by country
- Leading directors contributing to the content library
These visualizations allow users to explore how different factors influence content patterns and audience targeting.

Page 4 – Data Insights
This page summarizes the key observations derived from the analysis. It highlights important trends and patterns identified in the dataset, helping translate raw data into actionable insights that can inform strategic decision-making.

Page 5 – Business Strategy
Based on the insights generated from the analysis, this page proposes strategic recommendations that could potentially:
- increase revenue
- improve subscriber growth
- reduce subscriber churn

Estimated business impacts are derived using metrics from the Netflix Annual Report 2021. The projections are based on scenario modeling and analytical assumptions, and are intended to illustrate potential strategic outcomes rather than exact forecasts.
